# Blackbeard.js Documentation

#### Installation
Install `gitbook` command:
```
npm i gitbook-cli -g
```
Install npm dependencies:
```
npm install
```

Install gitbook plugins (in `docs` dir):
```
gitbook install
```

#### Build
```
gitbook build
```

#### Serve
```
gitbook serve
```

#### Publish
```
npm run publish
```
